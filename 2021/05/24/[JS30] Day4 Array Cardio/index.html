2. map()

第二題的練習為，將 inventors 內的 first 與 last 組合成一個陣列。

這裡會和forEach一起做比較。

map

map 會 return 新的陣列。

ForEach

foreach 不會回傳陣列，所以需要自己設一個陣列，再將資料推進去。

假設，今天要對陣列做一件事情，沒有要產生新的內容，就要使用 forEach。

如果希望是產生一個對應的新陣列，這時候就可以用map。

最大的差別就是，有沒有產生一個新的陣列。
