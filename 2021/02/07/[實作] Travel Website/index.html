CSS

font-size: 62.5%，1rem will be 10px。

各瀏覽器預設字型字號為16px，16px 相當於 1rem。

當我們定義 10px / 16px = 0.625 = 62.5% 時，瀏覽器的預設字型為 16px * 62.5% = 10px。

此時，10px 相當於 1rem。

  1. flex: 1 1 40rem，flex: grow shrink basis。flex-grow不長大，flex-shrink不縮小，flex-basis基本標準。

  2. 直接套給 logo 的字體,font-weight可以在下載字體時設定。

  1. flex-wrap:wrap，nav 的標籤會自動 Jump Down。什麼是 flex wrap

  1. flex: 1 1 40rem;，只要視窗空間小於 40rem，就會馬上 Jump Down。

  2. 讓 nav bar 保持在上方，position: sticky;，top: 0;，z-index: 3;

City Section CSS 解析

  1. overflow: hidden;，不讓 cloud 跑出去，加在根元素 section 上。
  2. text-decoration: underline;，不一定每個瀏覽器都支援。
  3. text-decoration-thickness: 0.5rem，加粗底線。

  1. 在字上面加上顏色，chrome 需要在前面加上-webkit 私有前綴才可以使用。

City Section CSS

Cloud Animation

  1. transform，位移。
    translate(mx,my)，元素以參考點為中心，X 軸位移 mx 距離，Y 軸位移 my 距離。
  2. animation 屬性，normal，reverse，alternate，reverse-alternate。
    參考網站
  3. infinite為重複播放。
  4. @keyframe製作動畫。

Media Query

  1. -webkit-text-fill-color: white;，讓字體變白色。
